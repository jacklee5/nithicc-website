Broken Physics

Broken physics is a 3D platformer meant to highlight the occasionally faulty physics that occurs when a novice developer attempts to implement the physics engine into a game. While the game only has 3 levels currently, they should be hard enough to keep you busy for a while. More levels are expected to be created soon.
